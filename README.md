💉 Vaccination Portal
A user-friendly web application that allows users to sign up, book vaccination appointments, and get updates about their vaccination status. Built using HTML, CSS, and JavaScript with modular component-based design.

🌟 Features
🔐 User Registration (Sign Up & Login)

📅 Appointment Booking Page

✅ Booking Confirmation Page

🧾 Dashboard to track appointments

📄 Vaccine Status Information

📚 FAQ and Information Section

🧭 Clean Navigation Bar and Footer Components

📱 Fully Responsive UI

🛠️ Tech Stack
HTML5 – Structure of the web pages

CSS3 – Styling the components (modular: footer.css, navbar.css, etc.)

JavaScript (Vanilla) – Form handling, dynamic content, navigation

React Components (optional) – Some modular components like App.js, Navbar.js, etc. resemble a component-based design
